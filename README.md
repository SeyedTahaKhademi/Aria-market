# 🇮🇷 ۱. داکیومنت به زبان فارسی (Persian)

### معرفی پروژه آریا مارکت
«آریا مارکت» یک پلتفرم تجارت الکترونیک مدرن، مینی‌مالیستی و پیشرفته است که امکان خرید نقدی و اقساطی بدون ضامن کالاهای لوکس دیجیتال، مد، زیبایی و لوازم خانگی را برای کاربران فراهم می‌کند. این پروژه با داشتن معماری تمام‌فلزی فرانت‌اند ری‌اکت و سرور قدرتمند اکسپرس، تجربه کاربری روان، انیمیشن‌های زنده و تعاملی غنی و یک دستیار خرید مجهز به هوش مصنوعی (AI Shopper) مبتنی بر مدل‌های زنده Gemini را در دسترس قرار می‌دهد.

### امکانات کلیدی پلتفرم
1. **کاتالوگ پویا و هوشمند کالاها:** تفکیک هوشمند محصولات در دسته‌های طلا، گجت، مد چرمی، محصولات زیبایی لوکس و لوازم خانگی مدرن با فیلتر آنی و جستجوی چندبعدی.
2. **سیستم درخواست اقساط آنی:** شبیه‌ساز هوشمند ارزیابی وضعیت اعتبار سنجی و درآمدی متقاضیان، محاسبه‌گر اتوماتیک درصد پیش‌پرداخت، مبالغ اقساط ماهانه بر اساس ماه انتخابی و صدور فوری کد رهگیری معتبر خریدار.
3. **پنل جامع فروشندگان (Vendor Dashboard):** امکان ثبت‌نام آسان برای توزیع‌کنندگان، ثبت کدهای ملی، بارگذاری اسناد فیزیکی و مانیتورینگ آنلاین میزان درآمد و وضعیت تایید فروشگاهی توسط مراجع مالی.
4. **دستیار خرید هوشمند پیشرفته (AI Shopper):** یک چت‌بات بسیار عمیق و کارآمد که پس از فعال‌سازی با API Key کاربر بر روی سرور، دسترسی کاملی به پایگاه داده کالاها پیدا کرده است. این بات می‌تواند مشخصات دقیق محصولات را مقایسه کند، راهنمایی برای بودجه‌بندی اقساطی ارائه کند و بهترین پیشنهاد را برای نیاز مشتری تجویز نماید.
5. **وبلاگ تخصصی و مقالات مالی:** مقالات غنی علمی پیرامون حفظ ارزش پول ملی در بازار تورمی، اصول ست‌های زیبایی چرمی، روتین‌های مراقبتی پوست و اصول مینیمالیسم در چیدمان منزل.

# 🇬🇧 2. English Documentation (English)

### Introduction to Aria Market
"Aria Market" is a high-performance, minimalist, and state-of-the-art e-commerce ecosystem that bridges the gap between premium retail merchants and consumers seeking flexible card-free, collateral-free installment options. Written entirely in TypeScript utilizing React on Vite for an eye-watering client-side experience and fully backed by an Express production server, it introduces an elegant workflow with a dedicated AI Shopping Guide empowered by OpenAI/Gemini systems.

### Core Architecture Features
1. **Interactive Multi-Category Storefront:** Sleek, micro-animated item cards detailing technical specifications, vendor origins, rating aggregates, and installment eligibility flags.
2. **Dynamic Installment Processor:** Interactive sliding controls allowing consumers to configure terms (up to 18 months), custom down payments, check salary limits, verify identity using Iranian National Code validation, and instantly track application pipelines with secure hash codes.
3. **Advanced Vendor Onboarding Center:** Complete self-service terminal for merchant brands, empowering vendors to track active registration pipelines, query gross historical revenues, and input structural inventory details.
4. **Server-Side AI Shopper:** Fully integrated with the latest `@google/genai` API SDK to execute context-aware conversations. It processes live database records (`db.json`) on the fly to generate semantic search recommendations and formulate customized budget plans according to the buyer’s spending capacity.
5. **Content-Rich Knowledge Hub:** High-fidelity articles outlining strategies for modern interior design, smart budget allocation in inflationary economies, and delicate care techniques for genuine leather commodities.

# 🇨🇳 3. 中文文档 (Chinese)

### 美亚市场 (Aria Market) 简介
“Aria Market（阿尔山市场）”是由 **Seyed Taha Khademi** 精心研发打造的下一代极简主义免保弹性分期购物电商平台。该项目结合了先进的端到端 TypeScript 前后端架构，前端采用 React (Vite) 提供电影级别的流畅动效，后端由高速精干的 Express 服务器支撑，深度对接谷歌最新官方 Gemini AI API 核心引擎，给用户带来无缝、尊贵、智能的数字消费享受。

### 核心功能板块
1. **极致视觉产品库:** 高保真卡片式设计，支持即时极速检索、模糊音过滤、多维度分类导航（数字科技、精奢皮革皮草、顶级香氛护肤、高尚家居生活）。
2. **智能化分期测算引擎:** 用户可以无缝调节分期期数（最高18个月）并自定义首付比例。系统通过自主设计的拟真评分算法，智能验证身份证和薪资标准，计算每期精确还款额，极速核发跟踪代码。
3. **集成多商户卖家控制台:** 允许独立零售品牌提交企业资质、国家识别号进行合规审批，并提供包含成交总额（GMV）、出货比例、销售评级的可视化看板。
4. **全集成 Gemini AI 导购客服:** 由服务器端极其强大的自然语言引擎注入。不仅懂得像人类一样亲切闲聊，更能像资深业务经理一样实时读取后台数据库，比较同类竞品性能，并为顾客量身定制还款组合。
5. **前沿美学与泛财经博客:** 发表针对高品质生活美学的精致文稿，如通货膨胀防守消费心理、牛皮大衣恒温保养工艺、临床玻尿酸修护及现代室内空间留白艺术。

# 🇦🇪 ۴. القسم الرابع: وثيقة باللغة العربية (Arabic)

### نظرة عامة على "آريا ماركت"
"آريا ماركت" (Aria Market) عبارة عن منصة ذكية وحديثة ومتكاملة للتجارة الإلكترونية والتسوق بالتقسيط الميسّر بدون كفيل أو فوائد خفية، تم تصميمها وهندستها ببراعة استثنائية بواسطة المطور **سيد طه خادمي**. تقدم المنصة واجهة مستخدم فائقة الرقي لتمكين المستهلكين من شراء السلع الرقمية الفاخرة، والجلود الأصلية، ومستحضرات التجميل الراقية، والأجهزة المنزلية، معززة بمستشار مالي ذكي يعتمد على خوارزميات الذكاء الاصطناعي التوليدي من جوجل Gemini.

### الميزات الوظيفية الأساسية
1. **كتالوج سلع متفاعل وقابل للتكيف:** تصفح انسيابي وسلس مع حركة فيزيائية وتأثيرات بصرية راقية لجميع المنتجات، مصحوبة بالمواصفات الفنية ومراجعات المشترين وتفاصيل البائعين.
2. **محاكي الأقساط الائتماني الذكي:** حاسبة متطورة تتيح للمستخدم إدخال الدخل الشهري، اختيار مدة السداد (تصل إلى ۱۸ شهراً)، ونسبة الدفعة الأولى، ليقوم النظام فوراً بتقييم الجدارة الائتمانية وتقديم رمز التتبع الائتماني الفريد.
3. **بوابة التجار المعتمدين (Vendor Hub):** منصة شاملة للبائعين والشركاء لتسجيل علاماتهم التجارية، والتأكد من هوياتهم الوطنية الرسمية، بالإضافة إلى إدارة إيراداتهم الإجمالية وحالات اعتماد حساباتهم التجارية.
4. **مساعد التسوق الذكي بالذكاء الاصطناعي (AI Shopper):** واجهة محادثة ذكية متكاملة تعمل على الخادم (Server-side) بشكل آمن تماماً، تقرأ محتويات قاعدة بيانات المنتجات وتقارنها، وتقدم اقتراحات شراء ذكية بناءً على قيود ميزانية المستخدم والقدرة الائتمانية.
5. **مدونة المعرفة المالية والجمالية:** مقالات عميقة مصممة لدعم ثقافة المستهلك، تركز على الحفاظ على قيمة رأس المال في فترات التضخم، وفن التنسيق المنزلي البسيط، والعناية بالمنتجات الجلدية الفاخرة.

# ⚙️ Technical Stack & Configuration Guide

### 🛠️ Core Technology Stack
| Layer | Technologies Used |
| :--- | :--- |
| **Client Frontend** | React, Tailwind CSS (Custom Theme), Framer Motion, Lucide Icons, TypeScript |
| **Server Backend** | Express, Node.js, Tsx Server Engine, Dotenv Security System, TypeScript |
| **Artificial Intelligence** | Google GenAI SDK (`@google/genai`) - model `gemini-2.5-flash` |
| **Storage Layer** | Relational-Simulated File Database System (`db.json`) |

### 🔑 Environment Configuration (`.env`)
The system expects host configurations and API tokens written in standard environment format. It secures keys strictly on the server-side proxy (`/api/ai-chat`), protecting confidential tokens from ever reaching public client-side browser storage (DevTools inspects).

```env
# Server Configuration and Security Rules
GEMINI_API_KEY="AQ.Ab8RN6Ig1MhgbfLdixw80uiv...[SECURE_API_KEY]"
APP_URL="https://ais-dev-...run.app"
PORT=3000
```

### 🚀 Production Deployment Commands
This platform is certified and compliant with modern enterprise orchestration containers (e.g., Cloud Run, Kubernetes, Docker deployments).
1. **Initialize Base Manifest and Build Assets:**
   ```bash
   npm run build
   ```
2. **Launch the Consolidated Server Engine:**
   ```bash
   npm run start
   ```
